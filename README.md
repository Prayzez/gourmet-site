# Gourmet Camp

A simple gourmet-themed static website created with HTML and CSS.

This project is a practice website for a gourmet information site. It features a clean landing page with a hero image, popular food categories, user reviews, and a sidebar search form.

## Features

- Responsive page structure with HTML and CSS
- Header navigation menu
- Hero image section
- Popular food genre section
- User review section
- Keyword search form in the sidebar
- Image assets for categories, icons, and branding

## Technologies Used

- HTML5
- CSS3

## Page Contents

### Header
The page includes a logo and a navigation menu with links such as:

- あなたにおすすめ
- ジャンルで選ぶ
- 雰囲気で選ぶ
- マイページ

### Main Visual
A large hero image is displayed at the top of the page.

### Popular Genres
Users can browse popular food categories such as:

- 中華
- 和食
- フレンチ
- イタリアン
- スイーツ

### User Reviews
The page includes sample user reviews with profile icons, user names, locations, and comments.

### Search Sidebar
A keyword search form is included in the sidebar for searching by terms such as area or food type.

## How to Use

1. Download or clone the repository
2. Open `gourmet_site.html` in your web browser
